# პროექტი: საჩუქრების პროგნოზირება (Tip Prediction)

## 📋 პროექტის აღწერა

ეს პროექტი იყენებს მანქანური სწავლების ალგორითმებს რესტორნის საჩუქრების მონაცემების ანალიზისთვის. პროექტი მოიცავს როგორც რეგრესიულ ანალიზს (საჩუქრის თანხის პროგნოზირებისთვის), ასევე კლასიფიკაციას (საჩუქრის კატეგორიზაციისთვის).

## 🎯 მიზანი

- **რეგრესია**: საჩუქრის ზუსტი თანხის პროგნოზირება სხვადასხვა ფაქტორების საფუძველზე
- **კლასიფიკაცია**: საჩუქრის კატეგორიზაცია (მაღალი/დაბალი) საშუალო მნიშვნელობის მიხედვით

## 📊 მონაცემთა ნაკრები

პროექტი იყენებს `tip.csv` ფაილს, რომელიც შეიცავს შემდეგ ინფორმაციას:

- `total_bill`: ანგარიშის საერთო თანხა
- `tip`: საჩუქრის თანხა
- `sex`: სტუმრის სქესი
- `smoker`: არის თუ არა სტუმარი მწეველი
- `day`: კვირის დღე
- `time`: დღის დრო (Lunch/Dinner)
- `size`: ჯგუფის ზომა

**მონაცემთა სტატისტიკა**: 243 სტრიქონი, 9 სვეტი (გაწმენდის შემდეგ)

## 🛠️ გამოყენებული ტექნოლოგიები

- **Python 3.x**
- **pandas** - მონაცემთა დამუშავებისთვის
- **scikit-learn** - მანქანური სწავლების ალგორითმებისთვის
- **Jupyter Notebook** - ინტერაქტიული ანალიზისთვის

## 🤖 გამოყენებული მოდელები

### რეგრესიული მოდელები:

1. **Lasso Regression** (R²: 0.518)
2. **Decision Tree Regressor** (R²: 0.266)
3. **Random Forest Regressor** (R²: 0.493)

### კლასიფიკაციური მოდელები:

1. **Logistic Regression** (Accuracy: 77.6%)
2. **Decision Tree Classifier** (Accuracy: 61.2%)
3. **K-Nearest Neighbors (KNN)** (Accuracy: 73.5%)

## 📈 შედეგები

### რეგრესიის შედეგები:

- **Lasso მოდელი** აჩვენა საუკეთესო შედეგები R² = 0.518-ით
- **Random Forest** მოდელმა დააფიქსირა R² = 0.493
- **Decision Tree** მოდელი ყველაზე სუსტი აღმოჩნდა (R² = 0.266)

### კლასიფიკაციის შედეგები:

- **Logistic Regression** აჩვენა ყველაზე მაღალი სიზუსტე (77.6%)
- **KNN** მოდელმა დააფიქსირა 73.5% სიზუსტე
- **Decision Tree** კლასიფიკატორი აჩვენა 61.2% სიზუსტე
